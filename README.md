# flask_SP

A simple secured flask application. In which a user need to first register into the application and has to wait until his account has been activated by the admin.
Admin and user can only upload document with .pdf, .doc, .jpeg, .png, .txt extension.
A user can see all the documents of his group. Admin can see all the contents.
A user can delete his content. Admin can delete any content.
Admin and user can download any content.
